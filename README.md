## 📲 How to Install (Simple Guide)

### ✅ TERMUX METHOD (Android)

1. 📱 Install **Termux** from Play Store  
2. 🧰 Inside Termux: `pkg install nodejs git`  
3. 🔽 Clone project: `git clone https://github.com/Toxicant1/B.E.L.T.A.H-`  
4. 📂 Enter: `cd B.E.L.T.A.H- && npm install`  
5. 📸 Scan WhatsApp QR when prompted  
✅ Done. Beltah is live 🟢

---

### ✅ TAMAX METHOD (No PC Needed)

1. 🔧 Download **Tamax** terminal  
2. 📁 Upload this bot folder into Tamax  
3. 🔁 Run `npm install`  
4. ▶️ Start: `node index.js`  
5. 📸 Scan QR and enjoy the vibes  

✅ Fully offline, powerful and stylish
