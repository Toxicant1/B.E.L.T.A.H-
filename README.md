📲 How to Install

✅ TERMUX (Android method)

1. Install Termux from Play Store


2. Inside Termux, andika:

pkg install nodejs git



3. Clone project:

git clone https://github.com/Toxicant1/B.E.L.T.A.H-.git



4. Open folder:

cd B.E.L.T.A.H- && npm install



5. Start bot:

node index.js



6. Scan WhatsApp QR code and enjoy 🟢



✅ Done. Beltah iko online 💫


---

✅ TAMAX METHOD (No PC Needed)

1. Fungua app ya Tamax (like Termux ya mtaa)


2. Upload project ya bot kwa Tamax 📂


3. Run: npm install


4. Then: node index.js


5. QR itaonekana, scan tu kama kawaida



✔️ Fast. Light. Built for mobile 🔥
