💡 What is Beltah?

> B.E.L.T.A.H isn’t just another WhatsApp bot 😎
It’s stylish, offline-ready, emoji-friendly, and AI-powered.
Built for Android users using Termux or Tamax — easy to run, no coding needed.

🧠 AI Chat (Romantic, Swahili Gen Z, Savage replies)

📲 How to Install

🟢 Option 1: TERMUX (Android)

1. Install Termux from Play Store


2. Run these in Termux:

pkg install nodejs git
git clone https://github.com/Toxicant1/B.E.L.T.A.H-
cd B.E.L.T.A.H- && npm install
node index.js


3. Scan the QR when prompted
✅ You're in!




---

🔵 Option 2: TAMAX (No Laptop Needed)

1. Install Tamax Terminal on Android


2. Upload this bot folder into Tamax


3. Run:

npm install
node index.js


4. Scan the QR code
✅ Beltah is now active!
